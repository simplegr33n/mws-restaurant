# MWS - Stage 3

## To Install:

**Gulp** is used as the default task runner. Run the following commands to start the project:

**in /_server/ dir:**
```
npm install
node server
```
to run the backend server.


**then, in base dir:**
```
npm install
npm start
```
to run the front end.

The application will be running on http://localhost:8000/
